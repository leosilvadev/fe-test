Endpoints
Show all the movies by query search:
GET https://sbot-fe-test.herokuapp.com/api/v1/movies?query={query}
The result is one movie with a listening_token in the body, you should use this token to listen to new movies for your search
Websocket usage
After try to find for a movie, the api will start to publish the results on the a ws channel with your listening_token:
Ex.: if you was searching for rambo.

Then you should listen in channel: movies.{listening_token}

Important: backend is using socket.io, so it is just easier for you to integrate through socket.io client library :)
